Admit Card Management System (Laravel)
1. Overview
The Admit Card Management System (ACMS) is a Laravel-based web application for managing school student data, exams, fees, and admit card generation. It supports Nursery to Class 12, multiple exam types (1st, 2nd, 3rd term, custom exams), and role-based user access.
